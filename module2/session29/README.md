# HR Policy RAG System

This project is a simple Retrieval-Augmented Generation (RAG) application built using Gemini embeddings and ChromaDB.  
It retrieves relevant HR policies from a vector database and generates grounded responses using Gemini LLM.  
The project also compares responses generated with and without RAG.  

## How to Run

1. Install dependencies using `pip install -r requirements.txt`
2. Add your Gemini API key
3. Run the project using `python hr_policy_rag.py`
